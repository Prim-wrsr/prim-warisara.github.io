üîç Project Overview:


This project analyzes a bank's loan portfolio to help monitor performance and make smarter lending decisions. Using SQL for data analysis and Tableau for visualization, I created interactive dashboard to explore loan trends, repayment behavior, and borrower profiles.

Project Objectives:

  • Track loan application trends and funded amounts.
  • Monitor repayments and calculate average interest rates.
  • Evaluate borrower financial health using debt-to-income ratio (DTI).
  • Identify the ratio of good loans vs bad loans.
  • Visualize trends by month, region, loan purpose and more.

Key Findings:

  • Most loan applications were for debt consolidation and credit card refinancing.
  • Borrowers with 10+ years of employment had the lowest default rate.
  • While 36-month loans make up the majority of total loans, 60-month loans carry greater risk and higher funding per borrower

    • Borrowers with 60-month loans received an average funded amount of $15,738, compared to $9,669 for 36-month loans.
    • 22.34% of 60-month loans were Charged Off, while only 10.71% of 36-month loans were.
    • 60-month borrowers were more than twice as likely to default compared to 36-month borrowers.

      • Lenders may consider implementing tighter approval criteria or risk-based pricing to reduce potential losses.

üí¨ Recommendations:

  • Consider tightening approval criteria for long-term loans to reduce risk exposure.
  • Use risk-based pricing (e.g., higher interest rates) for loans with higher default probability.

Sample SQL Queries:


                    --Total Loan Applications
                    SELECT COUNT(id) AS Total_Applications 
                    FROM bank_loan_data;
                

                    -- MTD Loan Applications
                    SELECT COUNT(id) AS Total_Applications 
                    FROM bank_loan_data
                    WHERE MONTH(issue_date) = 12;
                

                    -- PMTD Loan Applications
                    SELECT COUNT(id) AS Total_Applications 
                    FROM bank_loan_data
                    WHERE MONTH(issue_date) = 11;
                

                    --Total Funded Amount
                    SELECT SUM(loan_amount) AS Total_Funded_Amount 
                    FROM bank_loan_data;
                

                    -- MTD Total Funded Amount
                    SELECT SUM(loan_amount) AS Total_Funded_Amount 
                    FROM bank_loan_data
                    WHERE MONTH(issue_date) = 12;
                

                    -- PMTD Total Funded Amount
                    SELECT SUM(loan_amount) AS Total_Funded_Amount 
                    FROM bank_loan_data
                    WHERE MONTH(issue_date) = 11;
                

                    -- Total Amount Received
                    SELECT SUM(total_payment) AS Total_Amount_Collected 
                    FROM bank_loan_data;
                

                    -- PMTD Total Amount Received
                    SELECT SUM(total_payment) AS Total_Amount_Collected 
                    FROM bank_loan_data
                    WHERE MONTH(issue_date) = 11;
                

                    -- Average Interest Rate
                    SELECT AVG(int_rate)*100 AS Avg_Int_Rate 
                    FROM bank_loan_data;
